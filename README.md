# README

## Usage

### Heroku
- 後台入口：https://ac-restaurant-forum-yameday.herokuapp.com/
- 可使用 root@example.com 登入，密碼 12345678

### Git Clone Step

```
$ git clone https://github.com/yameday/restaurant_forum.git
$ bundle install
$ rails db:migrate
$ rails db:seed
$ rails dev:fake_restaurant
$ rails dev:fake_user
$ rails dev:fake_comment
```
